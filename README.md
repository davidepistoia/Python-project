ReFresh Food Manager 🍎
Python Project | start2impact University


📌 Vision & Mission

Every year, millions of tons of perfectly edible food are wasted globally. ReFresh Food is a (hypothetical) startup founded in Milan in 2023 with a clear goal: to transform this waste into opportunity. We recover unsold products or those with aesthetic defects and turn them into delicious, sustainable meals.

🎯 Project Objective

The goal of this project was to develop a management software that allows the ReFresh Food team to catalog and manage recipes created in our cooking lab. The system optimizes the use of recovered ingredients and facilitates recipe discovery based on current product availability.

🚀 Key Features

The application features an intuitive Graphical User Interface (GUI) to handle:

Recipe Management: Add, view, and delete recipes (including name, category, preparation time, and ingredients).

Smart Search: A quick text filter that scans through names, categories, and ingredients.

Advanced Filtering: Narrow down results by maximum preparation time, specific category, or a key ingredient.

Similarity Suggester: An algorithm that suggests alternative recipes sharing at least 2 ingredients with the selected one—ideal for optimizing stock usage.

Statistical Analysis: Automatically calculates total recipe count, average preparation times, and distribution across categories.

Data Persistence: All data is automatically saved in JSON format to ensure no information is lost when the app is closed.

System Logging: Integrated logging tracks all major operations (additions, deletions, errors) for system monitoring.

🛠️ Tech Stack
Python 3

Tkinter: For the Graphical User Interface (GUI).

JSON: Used as a lightweight local database.

Logging & Collections: For operational tracking and data analysis.

OOP (Object-Oriented Programming): The code is structured into classes (Manager and App) for high maintainability and scalability.
